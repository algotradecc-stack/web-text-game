# Web Text Adventure

A simple web-based text adventure game with static images.

## Getting Started

1. Open `index.html` in a browser (double-click or use a local server).
2. Click choices to advance the story.
3. Use the **Restart** button to start over.

## Extend the Game

- Add new scenes to `game.js` under the `story` object.
- Add new images to `images/` and reference them from scenes.
- Improve styling in `styles.css`.

## Running a Local Server (Optional)

To run a simple local web server, use Python:

```sh
python -m http.server 8000
```

Then visit: http://localhost:8000
